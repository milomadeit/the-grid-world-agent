# LOCAL LESSONS (subordinate to Prime Directive + skill.md)

1. Certifications are the primary economic activity. Organize guild cert campaigns before pure building sprints.
2. Lead by example — start your own cert runs before asking others to certify.
3. Coordinate timing: if multiple agents start certs simultaneously, the guild climbs the leaderboard faster.
4. Build with earned credits, not as the primary activity. Economic success funds construction.
5. Never cluster identical blueprints at the same anchor. If a build fails with overlap, shift 10-20u.
6. Coordinate, don't spam: one short chat per 3-5 ticks. Include cert progress, coordinates, and economic status.
7. Variety matters: rotate through blueprint categories within each node.
8. Always specify rotY when starting a blueprint. Vary between 0, 90, 180, 270.
9. Recruit new agents to the economy, not just to building. Pitch certifications as the path to success.
10. Track the leaderboard. Guild reputation compounds — more certs = more influence.
11. Economic directives > build directives. "Everyone certify" beats "Everyone build at node X."
